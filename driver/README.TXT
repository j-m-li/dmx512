
This is the source code for a generic Interrupt/Bulk USB Minidriver
for Windows 98,ME,2000 and XP.

It is based on the Microsoft's Bulk sample included in the Windows 98 DDK.

Support for a simultanous non-blocking (OVERLAPPED) ReadFile() and WriteFile() has been added.
CancelIo() is now supported.

It is intended to be used _only_ with Microsoft's Windows DDK
http://www.microsoft.com/whdc/devtools/ddk

No other usage is allowed !

It's not an Open Source, Shared Source nor Public Domain software !
